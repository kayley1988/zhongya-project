📋 计算明细与公式

▶
落地总成本 C
C = 采购价 + 国内运输 + 国际运输 + 装卸费 + 清关费 + 保险 + 预备费 + 税费
C = ¥800,000 + ¥15,000 + ¥18,000 + ¥5,000 + ¥8,000 + ¥6,400 + ¥5,000 + ¥141,032 = ¥998,432
24个月运营成本 O
O = (维修预提 + 保险 + 管理费 + 场站费) × 24
O = (¥3,000 + ¥1,500 + ¥2,000 + ¥1,000) × 24 = ¥180,000
残值 S
S = 采购价 × 残值率
S = ¥800,000 × 50% = ¥400,000
满足毛利率的总收入 Rtotal
Rtotal = (C - S + O) / (1 - g)
Rtotal = (¥998,432 - ¥400,000 + ¥180,000) / 0.7 = ¥1,112,046
建议月租 Rmonth
Rmonth = Rtotal / (T × u)
Rmonth = ¥1,112,046 / (24 × 0.7) = ¥66,193

📜 租赁条款摘要

💰 押金

首次租赁需缴纳2个月租金作为履约保证金，租期结束无损坏/欠费情况下全额退还。

📅 最低租期

最低租期 3个月，不足3个月按3个月计费。长期租赁（12个月+）可享折扣。

⏱️ 超时计费

日租按8小时/天计算，超出部分按小时费率×1.5计费，夜间作业另加20%。

🔧 维修责任

日常保养：承租方负责
非人为故障：出租方负责
人为损坏：承租方赔偿

⚠️ 逾期处理

逾期7天内：日租金×0.1%滞纳金
逾期30天：有权收回设备并扣除押金

🛡️ 保险要求

承租方需购买第三者责任险，设备损失险由出租方统一投保并计入租金。

📊 两种回本口径说明

📊

总投资回本（项目资产口径）

看的是：设备净成本（含落地）+ 运营成本 能否在24个月被收入覆盖，并实现毛利30%

落地总成本 C ¥998,432
24个月运营成本 O ¥216,000
残值 S ¥400,000
需收入覆盖 ¥814,432

✅ 适用于评估项目整体盈利能力

💰

自有资金回本（股权现金口径）

看的是：实际掏出的现金（首付、保证金、垫付费用）何时靠净现金流回正

首付（融资模式） ¥299,530
每月：租金-还款-运营 +¥24,368
第24月：残值回收 +¥400,000
累计净现金流=0 第12个月

✅ 适用于评估资金回收速度和杠杆效果

💡 建议：融资模式下，自有资金回本通常更快；全款模式下，两种口径差异较小。

🔧 维修与大修规则说明

🔩

小修（日常保养）

每月预提
  • 发动机机油及滤芯更换
  • 液压油滤芯更换
  • 空气滤芯更换
  • 黄油润滑
  • 易损件更换（橡胶件、密封圈等）
  • 日常检修调试
建议预提： ¥1,500-3,000/月
⚙️

大修（重大维修）

按工时/月预提
  • 发动机大修（约6000工时）
  • 液压泵/马达大修（约4000工时）
  • 底盘件更换（链条/支重轮/引导轮）
  • 驾驶室/电气系统大修
  • 结构件焊接修复
建议预提： ¥2,000-5,000/月
