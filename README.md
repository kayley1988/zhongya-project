# 中亚跨境项目管理系统

## 系统概述

飞书风格的项目管理系统，用于管理中哈跨境机械设备租售项目的测算和跟踪。

## 功能特性

### 📋 项目列表页 (list.html)
- 飞书风格的数据表格
- 支持按状态、区域筛选
- 支持搜索项目名称、客户
- 显示关键KPI指标（GM1、PB1、运输占比）
- 支持批量测算、导出、删除
- 分页浏览

### 📊 项目测算页 (project.html)  
- 完整的测算参数配置（7大类）
- 3种毛利率指标（GM1/GM2/GM3）
- 2种回本指标（PB1/PB2）
- 可视化图表分析
- 版本管理（保存/加载/对比）
- PDF报告导出

### 💾 数据存储 (IndexedDB)
- 本地浏览器存储
- 支持离线使用
- 数据持久化

## 文件结构

## 快速开始

### 方法1：直接打开
直接在浏览器中打开 `list.html` 或 `index.html`

### 方法2：本地服务器
```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 VS Code Live Server 插件
```

然后访问 `http://localhost:8080`

## 使用流程

1. **进入列表页** - 查看所有项目概览
2. **新建项目** - 点击"新建项目"按钮
3. **配置参数** - 在项目详情页填写测算参数
4. **执行测算** - 点击"立即测算"
5. **保存版本** - 点击"保存版本"记录当前配置
6. **导出报告** - 点击"导出报告"生成PDF

## KPI指标说明

| 指标 | 含义 | 达标线 |
|------|------|--------|
| GM1 | 经营毛利率（收入-可变成本） | ≥30% |
| GM2 | 含摊销毛利率（GM1-折旧） | ≥30% |
| GM3 | 全口径毛利率（GM2-资金成本） | ≥30% |
| PB1 | 项目回本期（不考虑融资） | ≤24月 |
| PB2 | 股东回本期（考虑融资结构） | ≤24月 |

## 技术栈

- 纯前端实现（HTML/CSS/JavaScript）
- IndexedDB 本地数据库
- Chart.js 图表库
- 响应式设计

## 注意事项

1. 数据存储在浏览器本地，清除浏览器数据会丢失
2. 建议定期导出重要项目数据
3. 首次打开会自动生成示例数据
