# 中亚项目部署操作手册（网页版）

## 📋 准备工作

确保你有GitHub账号，如果没有：
1. 访问 https://github.com/signup
2. 注册一个免费账号

---

## 🚀 第一步：创建GitHub仓库

1. 登录GitHub后，访问：**https://github.com/new**

2. 填写信息：
   - **Repository name**（仓库名）：`zhongya-project`
   - **Description**（描述）：中亚跨境项目管理系统
   - 选择 **Public**（公开）
   - ✅ 勾选 "Add a README file"
   - 点击绿色按钮 **"Create repository"**

---

## 📤 第二步：上传文件

1. 进入刚创建的仓库页面

2. 点击 **"Add file"** → **"Upload files"**

3. 打开文件夹：`d:\test\2025.12.17中亚\2025.12.17中亚\2025.12.17中亚\中亚\中亚`

4. **全选所有文件**（Ctrl+A），拖拽到GitHub网页的上传区域
   - 包括所有 .html、.css、.js、.md 文件
   - 包括 api/、database/、lib/ 等文件夹

5. 等待上传完成（可能需要1-2分钟）

6. 在底部 "Commit changes" 填写：
   - **Commit message**：`初始版本`
   - 点击绿色按钮 **"Commit changes"**

---

## 🌐 第三步：开启GitHub Pages

1. 在仓库页面，点击顶部的 **"Settings"**（设置）

2. 在左侧菜单找到 **"Pages"**

3. 在 "Source" 部分：
   - Branch 选择：**main**
   - Folder 选择：**/ (root)**
   - 点击 **"Save"**

4. 等待30秒，刷新页面，会看到绿色提示：
   ```
   Your site is live at https://你的用户名.github.io/zhongya-project/
   ```

5. **复制这个网址**，等下要用

---

## 🎯 第四步：测试访问

1. 打开浏览器，访问：
   ```
   https://你的用户名.github.io/zhongya-project/list.html
   ```

2. 如果能看到项目列表页面，说明部署成功！

---

## 📱 第五步：配置企业微信

### 5.1 创建自建应用

1. 登录企业微信管理后台：https://work.weixin.qq.com/

2. 点击 **"应用管理"** → **"应用"** → **"自建"**

3. 点击 **"创建应用"**

4. 填写应用信息：
   - 应用名称：`中亚项目管理`
   - 应用Logo：上传一个图标（可以随便选一个）
   - 应用介绍：`跨境项目测算与管理系统`
   - 可见范围：选择需要使用的部门（或全公司）

5. 点击 **"创建应用"**

### 5.2 配置可信域名

1. 进入刚创建的应用详情页

2. 找到 **"网页授权及JS-SDK"** 部分

3. 点击 **"设置可信域名"**

4. 输入：`你的用户名.github.io`
   - ⚠️ 注意：只填域名部分，不要加 https:// 和路径

5. 会提示下载一个验证文件（类似 `WW_verify_xxx.txt`）

6. **下载这个文件**

### 5.3 上传验证文件

1. 回到GitHub仓库页面

2. 点击 **"Add file"** → **"Upload files"**

3. 上传刚才下载的验证文件 `WW_verify_xxx.txt`

4. 提交更改

5. 等待1分钟，GitHub Pages会自动更新

6. 回到企业微信，点击 **"确认"** 完成验证

### 5.4 设置应用主页

1. 在应用详情页，找到 **"应用主页"**

2. 填写：
   ```
   https://你的用户名.github.io/zhongya-project/list.html
   ```

3. 点击 **"保存"**

---

## ✅ 完成！

现在企业成员可以：

1. 打开企业微信APP
2. 进入 **"工作台"**
3. 看到 **"中亚项目管理"** 应用
4. 点击直接使用

---

## 🔧 如果遇到问题

**问题1：GitHub Pages 显示404**
- 等待5分钟再试
- 检查仓库是否设为Public
- 确认文件已上传成功

**问题2：企业微信打不开**
- 检查可信域名是否配置正确
- 确认验证文件已上传
- 用手机浏览器先测试网址能否打开

**问题3：页面显示不正常**
- 检查是否所有文件都上传了
- 特别是 api/、database/、lib/ 文件夹
