## 系统简介

### 系统功能概览

本系统是一套专为工程机械行业打造的价格分析工具，主要功能包括：

✅ **智能价格分析**
- 选中任意设备型号，一键获取多维度价格信息
- 新机价格（官方指导价、经销商成交价）
- 二手机价格（按车龄分级：3-5年、5-8年、8年以上）
- 租赁价格（台班价、包月价、包年价）
- 区域价格差异对比

✅ **专业报告生成**
- 符合行业标准的PDF报告自动生成
- 包含：设备参数、价格分析、采购建议、风险提示
- 支持多种报告类型：综合分析、设备对比、市场趋势、区域价格

✅ **数据管理系统**
- Web管理后台查看和管理设备数据
- 热门设备追踪
- 价格历史趋势分析
- 报告下载和分享

✅ **中亚市场专项分析**
- 哈萨克斯坦、乌兹别克斯坦市场需求分析
- 物流成本影响评估
- 投资建议和风险提示

## 快速开始

### 第一步：安装油猴扩展

1. **Chrome浏览器**
   - 访问 [Chrome网上应用店](https://chrome.google.com/webstore)
   - 搜索 "Tampermonkey"
   - 点击"添加至Chrome"

2. **Firefox浏览器**
   - 访问 [Firefox Add-ons](https://addons.mozilla.org)
   - 搜索 "Greasemonkey" 或 "Tampermonkey"
   - 点击"添加到Firefox"

3. **Edge浏览器**
   - 访问 [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons)
   - 搜索 "Tampermonkey"
   - 点击"获取"

### 第二步：安装价格分析脚本

1. 点击浏览器工具栏的Tampermonkey图标
2. 选择"管理面板"
3. 点击左侧"+"按钮创建新脚本
4. 复制以下脚本内容并粘贴：

**脚本1：智能价格分析助手**
- 文件：`smart-machinery-price-analyzer.user.js`
- 功能：选中设备型号即可查看价格分析

**脚本2：PDF报告生成器**
- 文件：`machinery-report-generator.user.js`
- 功能：一键生成专业PDF报告

5. 点击"文件" → "保存"
6. 确认脚本已启用（开关为绿色）

### 第三步：开始使用

1. 访问任意包含工程机械信息的网页（如设备采购网站、论坛等）
2. 用鼠标选中设备型号（例如："徐工XCT25"、"50吨汽车起重机"）
3. 按快捷键 `Ctrl+Shift+M` 或右键选择"分析工程机械价格"
4. 等待分析窗口弹出，查看多维度价格信息

## 油猴脚本使用

### 价格分析功能

#### 启动方式

**方法1：快捷键（推荐）**
1. 选中设备型号文字
2. 按 `Ctrl+Shift+M`
3. 自动弹出分析窗口

**方法2：右键菜单**
1. 选中设备型号文字
2. 右键点击
3. 选择"🏗️ 分析工程机械价格"

**方法3：浮动按钮**
- 页面右下角有浮动按钮"📄 生成价格分析报告"
- 点击后可手动输入设备型号

#### 分析界面说明

分析窗口包含5个标签页：

**1️⃣ 综合分析**
- 设备基本信息（类型、吨位、规格）
- 核心技术参数（发动机、功率、臂长等）
- 价格摘要和建议购买时机
- 投资回报周期计算
- 同步数据到服务器按钮

**2️⃣ 新机价格**
- 官方指导价区间
- 经销商实际成交价
- 优惠空间分析
- 购买建议（时机、品牌推荐）

**3️⃣ 二手价格**
- 3-5年车龄价格区间
- 5-8年车龄价格区间
- 保值率计算
- 年折旧率分析
- 检查要点提示

**4️⃣ 租赁分析**
- 日租金（台班价）
- 月租金（包月价）
- 年租金（包年价）
- 租赁 vs 购买对比
- ROI分析图表

**5️⃣ 决策建议**
- 智能采购策略推荐
- 价格走势预期
- 风险提示
- 中亚市场专项分析（哈萨克斯坦、乌兹别克斯坦需求、物流成本）

#### 支持的设备型号

**汽车起重机**
- 徐工系列：XCT25, XCT50, XCT75, XCT100等
- 三一系列：STC250T, STC500等
- 中联系列：ZTC251V等
- 通用型号：25吨汽车起重机、50吨汽车起重机等

**其他设备类型**
- 系统支持通过吨位和类型匹配
- 示例："25吨"+"起重机" → 自动匹配对应数据
- 数据库持续更新中

#### 数据同步

点击"🔄 同步数据到 jinzhe.asia"按钮：
- 将分析结果上传到服务器
- 保存到您的个人数据库
- 可在Web后台查看历史分析记录
- 用于生成综合报告

## Web管理后台

### 访问后台

**访问地址**：`http://www.jinzhe.asia/admin/price-dashboard.html`

（如已配置HTTPS：`https://www.jinzhe.asia/admin/price-dashboard.html`）

### 后台功能

#### 1. 统计看板

顶部4个统计卡片：
- **设备总数**：系统收录的设备数量
- **今日查询**：当天使用脚本查询的次数
- **价格更新**：最近价格数据更新时间
- **报告生成**：本月生成的报告数量

#### 2. 设备价格库

**功能**：
- 查看所有设备的价格数据
- 搜索设备（按型号、品牌）
- 查看设备详情
- 添加新设备

**使用方法**：
1. **搜索设备**
   - 在搜索框输入关键词（如"徐工"、"25吨"）
   - 点击"🔍 搜索"
   - 结果实时更新

2. **查看详情**
   - 点击任意设备卡片
   - 弹出详情窗口
   - 显示完整参数和价格信息

3. **添加设备**
   - 点击"+ 添加设备"按钮
   - 填写表单：
     * 设备型号（必填）
     * 品牌（必填）
     * 设备类型（必填）
     * 吨位/规格
     * 新机价格区间（必填）
     * 备注
   - 点击"提交"

#### 3. 热门查询榜

**显示内容**：
- 近7天查询最多的设备
- 排名前10
- 显示查询次数和最后查询时间

**用途**：
- 了解市场热度
- 发现用户关注点
- 指导设备采购决策

#### 4. 报告导出

**报告类型**：

1. **综合价格分析报告**
   - 包含：参数、价格、分析、建议
   - 适用：全面了解单个或多个设备

2. **多设备对比报告**
   - 包含：横向对比表格、价格对比图表
   - 适用：比较不同品牌/型号

3. **市场趋势报告**
   - 包含：价格走势、预测分析
   - 适用：了解市场动态

4. **区域价格报告**
   - 包含：中亚市场专项分析
   - 适用：出口业务决策

**导出方法**：
1. 点击对应报告类型卡片
2. 系统自动生成PDF
3. 浏览器自动下载
4. 报告保存到服务器（可在后台查看历史）

## 报告生成与导出

### 报告结构

根据您提供的标准，生成的PDF报告包含以下章节：

#### 一、报告概述
- 1.1 报告目的
- 1.2 覆盖范围
- 1.3 价格维度说明

#### 二、核心设备信息汇总表
表格包含：
- 序号
- 设备品类
- 型号规格
- 核心技术参数
- 新机价格（官方指导价、经销商成交价）
- 二手机价格（3-5年、5-8年）
- 租赁价格（台班/包月/包年）
- 价格说明/备注

#### 三、分维度价格补充说明
- 3.1 新机价格专项说明表
  * 官方指导价
  * 经销商最低成交价
  * 批量采购价（≥3台）
  * 价格包含内容
  * 优惠政策（促销/分期）
  * 区域价格差异

- 3.2 二手机价格影响因素分析
  * 基准价格（3年车龄/优工况）
  * 车龄影响
  * 工况影响
  * 维保记录影响
  * 热门程度

- 3.3 租赁价格细分表
  * 台班价（8小时）
  * 包月价（含台班数）
  * 包年价
  * 附加费用（进出场/燃油/操作手）
  * 租赁最低期限
  * 押金要求

#### 四、参数与价格关联分析
- 4.1 核心参数对价格的影响规律
  * 起重量影响（每提升10吨，价格提升30%-50%）
  * 发动机型号影响（进口比国产贵8%-15%）
  * 配置升级影响（智能系统、副臂等）

- 4.2 同参数不同品牌价格对比
  * 品牌知名度影响
  * 市场保有量影响
  * 技术先进性影响

#### 五、结论与建议
- 5.1 采购建议
  * 短期使用：租赁方案
  * 长期使用：新机购买
  * 预算有限：二手机方案

- 5.2 价格谈判要点
  * 新机谈判策略
  * 二手机谈判要点
  * 租赁合同注意事项

- 5.3 风险提示
  * 二手机风险
  * 租赁价格波动
  * 新机价格影响因素

#### 六、附录
- 6.1 数据来源详情
- 6.2 设备型号补充说明
- 6.3 术语解释（台班价、残值率、工况等级）

### 报告编号规则

**格式**：`JXSB-JG-YYYY-XXX`

- JXSB：机械设备（拼音缩写）
- JG：价格（拼音缩写）
- YYYY：年份（如2025）
- XXX：流水号（001-999）

**示例**：`JXSB-JG-2025-001`

## 常见问题

### 1. 脚本无法启动

**问题**：选中文字后没有反应

**解决方法**：
1. 检查Tampermonkey扩展是否已安装
2. 检查脚本是否已启用（开关为绿色）
3. 刷新页面重试
4. 查看浏览器控制台（F12）是否有错误

### 2. 分析窗口显示"暂未收录该设备"

**问题**：设备型号未在数据库中

**解决方法**：
1. 尝试使用更通用的关键词（如"25吨汽车起重机"）
2. 在Web后台手动添加该设备
3. 联系管理员更新数据库

### 3. 价格数据不准确

**问题**：显示的价格与实际市场价差异较大

**解决方法**：
1. 价格仅供参考，以实际签约为准
2. 可在Web后台更新最新价格
3. 建议结合多个数据源综合判断

### 4. PDF报告无法生成

**问题**：点击生成报告按钮没有响应

**解决方法**：
1. 检查网络连接
2. 确认jsPDF库是否加载（控制台F12）
3. 检查浏览器是否阻止了下载
4. 清除浏览器缓存后重试

### 5. 数据无法同步到服务器

**问题**：点击同步按钮后提示失败

**解决方法**：
1. 检查网络连接
2. 确认API地址是否正确
3. 检查API密钥是否有效
4. 查看服务器日志排查错误

### 6. Web后台无法访问

**问题**：打开管理后台页面报错

**解决方法**：
1. 检查服务器是否正常运行
2. 检查Nginx/Apache配置
3. 查看服务器防火墙设置
4. 检查域名DNS解析

### 7. 设备添加失败

**问题**：在后台添加设备时提示错误

**解决方法**：
1. 检查必填字段是否填写完整
2. 确认价格格式是否正确（数字）
3. 检查数据库连接状态
4. 查看PHP错误日志

### 8. 报告中的中亚市场数据在哪里？

**问题**：找不到哈萨克斯坦、乌兹别克斯坦的分析

**解决方法**：
1. 在分析窗口切换到"决策建议"标签页
2. 查看"中亚市场分析"部分
3. PDF报告的"五、结论与建议"章节包含中亚市场内容

### 9. 如何批量导入设备数据？

**问题**：需要一次性添加多个设备

**解决方法**：
1. 准备JSON格式的设备数据
2. 使用API批量导入接口
3. 参考DEPLOYMENT.md中的批量导入说明
4. 或使用phpMyAdmin直接导入数据库

### 10. 历史报告在哪里查看？

**问题**：找不到之前生成的报告

**解决方法**：
1. 登录Web管理后台
2. 查看"报告生成"统计卡片
3. 点击查看历史报告列表
4. 报告文件保存在服务器 `/uploads/reports/` 目录

## 💡 使用技巧

### 技巧1：快速对比多个设备

1. 打开Excel或记事本
2. 列出要对比的设备型号
3. 逐一选中型号并按 `Ctrl+Shift+M`
4. 记录关键价格信息
5. 最后生成综合对比报告

### 技巧2：追踪价格变化

1. 定期查询同一设备
2. 数据会自动保存到服务器
3. 在Web后台查看价格历史趋势
4. 生成市场趋势报告分析走势

### 技巧3：利用报告进行谈判

1. 生成包含多个品牌的对比报告
2. 报告中包含市场均价和经销商成交价
3. 以报告数据为依据进行价格谈判
4. 争取接近或低于经销商最低成交价

### 技巧4：中亚出口决策

1. 查看设备的"决策建议"标签页
2. 重点关注物流成本估算
3. 结合中亚市场需求分析
4. 计算总成本和预期利润
5. 生成区域价格报告用于决策

### 技巧5：租赁 vs 购买决策

1. 查看"租赁分析"标签页
2. 查看"购买回本期"数据
3. 根据使用期限判断：
   - 使用期 < 回本期：建议租赁
   - 使用期 > 回本期：建议购买
4. 考虑资金占用和折旧因素

## 📄 附录

### 术语解释

**台班价**：按8小时工作制计算的单日租赁价格

**残值率**：设备使用年限后的剩余价值占原值的百分比

**工况等级**：设备使用状况评级，分为优、良、一般、差四个等级

**保值率**：二手设备价格占新机价格的百分比

**批量采购价**：一次性购买3台及以上设备的优惠价格

**进出场费**：设备从停放地点运输到施工现场的费用

**优工况**：设备使用良好，维保完整，外观整洁，性能正常

**良工况**：设备正常使用，有轻微磨损，性能基本正常

**一般工况**：设备使用较多，有明显磨损，需要维修
